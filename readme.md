## Requirements

- macOS 10.12+
- Xcode 10+
- Swift 4.2+


## Install

#### SwiftPM

```swift
.package(url: "https://github.com/wulkano/Aperture", from: "0.0.0")
```

#### Carthage

```
github "wulkano/Aperture"
```

#### CocoaPods

```ruby
pod 'Aperture'
```
